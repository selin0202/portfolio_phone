BOOTSTRAP

Media Query를 이용한 반응형 웹사이트 입니다.

이 웹사이트는 다양한 디바이스에서 최적화된 사용자 경험을 제공하기 위해 반응형 웹 디자인을 적용한 프로젝트입니다.

화면 크기에 따라 유동적으로 레이아웃이 조정되며, 모바일, 태블릿, 데스크탑 등 모든 환경에서 원활하게 동작합니다.

디바이스에 관계없이 사용자에게 일관된 레이아웃 구성을 제공하고, 다양한 화면에서 원활하게 동작하는 반응형 웹사이트를 구현하는 것이 목표였습니다.

Open Api를 이용한 도서사이트 입니다.

Open API를 활용하여 실제 데이터를 처리하고, 외부 서비스와 연동하는 방법을 익힐 수 있었습니다.

특히, KAKAO OPEN API의 사용법을 배우고, 이를 통해 다양한 책 정보를 동적으로 불러오는 방법을 구현하면서 데이터 처리 및 API 통합 경험을 쌓았습니다. 그와 동시에 jqeury로 슬라이더를 구현하여 사용자에게 보다 더 많은 도서의 정보를 볼 수 있도록 구현하였습니다.

API 연동의 기초부터 실전 활용까지 경험할 수 있었습니다. 향후 Open API를 활용한 프로젝트에서도 이와 같은 기술을 활용 할 수 있습니다.

JSON을 이용한 쇼핑몰 웹 사이트 입니다.

이 웹사이트는 JSON 파일을 통해 데이터를 관리하고, AJAX로 데이터를 가져와 페이지에 동적으로 렌더링하는 방식으로 개발했습니다.

JSON 데이터를 사용해보면서 콘텐츠를 쉽게 수정하고 추가할 수 있는 유연한 구조를 만들고자 했습니다.

이 프로젝트를 통해 JSON과 JavaScript의 상호작용에 대한 이해를 깊이 할 수 있었고, 데이터를 효율적으로 관리하는 웹 개발의 기본적인 흐름을 경험할 수 있었습니다.

React를 이용한 쇼핑몰 웹 사이트 입니다.

HTML과 비슷한 JSX 문법을 사용하여 레이아웃을 구성했고, 화면의 각 부분을 React 컴포넌트로 나누어 관리하기 쉽게 했습니다.

Redux를 통해 페이지의 상태를 중앙에서 관리하여 여러 컴포넌트 간에 데이터를 쉽게 공유할 수 있도록 했습니다. Redux의 store를 사용해 데이터를 저장하고, 필요한 컴포넌트에서만 상태가 변경되도록 하여 코드의 복잡성을 줄이는 데 중점을 두었습니다.

이번 프로젝트를 통해 React와 Redux의 기본적인 연동 방식을 배울 수 있어 나중에 프론트엔드 개발을 할 때도 큰 도움이 될 것이라 생각됩니다.
